File System Map Important Files

Filesystem map and high-value paths for threats/defense

Critical paths

/etc/shadow                                  # Hashed local passwords; offline cracking ⇒ full host compromise
/root/.ssh/                                  # Root private keys and trusts; immediate total control
/var/run/docker.sock                         # Root-equivalent Docker control; container breakout ⇒ host root
/etc/ld.so.preload                           # Force-load library into all processes; stealth persistence
/etc/sudoers                                 # NOPASSWD/wildcards ⇒ instant root; audit carefully
/etc/systemd/system/*.service                # Autostart root services; backdoor with Exec= payloads
/var/www/**/.env                             # App secrets (DB/API keys) in prod; instant data access

High-risk paths

/home/*/.ssh/                                # User SSH keys; lateral movement and trust pivot
/etc/ssh/sshd_config                         # Weak auth/ciphers; brute-force and backdoor risk
/etc/krb5.keytab                             # Kerberos service keys; AD lateral via SPN impersonation
~/.aws/credentials                           # Cloud keys; tenant-wide impact
~/.config/gcloud/*                           # GCP auth; project takeover
~/.azure/*                                   # Azure tokens; subscription lateral
/var/www/**/config.php                       # Hard-coded creds/salts; DB and admin panel pivot
/srv/*/settings.py                           # Framework secrets; admin tokens
/etc/ld.so.conf.d/*                          # Library path hijack opportunities
find / -perm -4000 -type f                   # SUID binaries; misconfigs ⇒ privesc
getcap -r /                                  # File capabilities; elevated syscalls without SUID

Medium-risk paths

/var/log/auth.log                            # Logins/sudo traces; detect misuse and brute-force
/var/log/syslog                              # System events; correlate execution times
/var/log/audit/audit.log                     # Syscall traces; file/exec monitoring
/var/log/journal/                            # Persistent systemd logs; tamper detection
/usr/share/wordlists/                        # Known lists (rockyou); inventory and integrity
~/.bash_history                              # Command trails; OPSEC leak, forensic value
~/.zsh_history                               # Same as above for zsh
/var/lib/dpkg/                               # What’s installed; identify suspicious packages
/etc/NetworkManager/system-connections/*     # Saved Wi-Fi/802.1x creds; network pivot
/etc/nftables.conf                           # Policy gaps enable ingress/exfil
/var/www/html                                # Default docroot; common webshell target
/tmp , /var/tmp                              # Staging for payloads; scan and clean
