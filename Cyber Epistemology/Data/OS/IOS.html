High-Value Paths Important Files

Filesystem map and high-value paths for threats/defense

Critical paths

/private/var/Keychains/keychain-2.db                     # System keychain DB; device & app secrets (file exists, access gated by SEP)
/private/var/mobile/Library/ConfigurationProfiles/*      # MDM/config profiles; trust, VPN, certs
/Library/LaunchDaemons/*.plist (jailbroken)              # Root persistence (launchd); verify owners/paths
/Library/LaunchAgents/*.plist (jailbroken)               # Global user-session persistence
/private/var/containers/Shared/SystemGroup/*             # System groups/shared containers; tokens/configs
/private/var/mobile/Library/Accounts/Accounts*.sqlite    # iCloud & account tokens (entitlement protected)
/private/var/managed/* (Supervised/MDM)                  # Managed settings/apps; policy scope

High-risk paths

/private/var/mobile/Containers/Data/Application/<UUID>/ # App sandboxes (Documents, Library, tmp)
/private/var/mobile/Containers/Shared/AppGroup/<UUID>/   # App Groups shared data between apps
/private/var/mobile/Library/Preferences/*.plist          # Per-app/system prefs; tokens/endpoints
/private/var/preferences/SystemConfiguration/*           # Wi-Fi, DHCP, DNS, proxies
/private/var/mobile/Library/Safari/*                     # History, bookmarks, tabs
/private/var/mobile/Library/WebKit/*                     # WebKit caches, WebsiteData
/private/var/mobile/Library/Keyboard/*                   # User dictionaries; sensitive words
/private/var/mobile/Media/DCIM/*                         # Photos/videos (exfil target)
/private/var/mobile/Library/MobileDocuments/*            # iCloud Drive synced data
/private/var/mobile/Library/CallHistoryDB/CallHistory.storedata   # Call history DB
/private/var/mobile/Library/SMS/sms.db                   # Messages DB (unencrypted on device; backup-encrypted)

Medium-risk paths

/private/var/mobile/Library/Logs/CrashReporter/*         # Crash logs; execution traces
/private/var/db/diagnostics/*                            # Unified logging tracev3 (via log show)
/private/var/mobile/Library/AddressBook/AddressBook.sqlitedb (older)  # Contacts (legacy path)
/System/Volumes/Data/*                                   # Writable data volume (modern iOS layout)
/private/var/tmp , /private/var/mobile/Library/Caches/*  # Staging/caches; loot & OPSEC leaks
/private/etc/hosts (jailbroken)                          # Traffic redirection if modified
/private/etc/ssh/sshd_config (jailbroken)                # Remote entry policy

Note: On stock iOS, access to many paths is restricted by sandbox, entitlements, and the Secure Enclave; full file access typically requires a device backup (iTunes/USB) or a jailbroken device.
