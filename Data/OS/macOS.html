High-Value Paths Important Files

Filesystem map and high-value paths for threats/defense

Critical paths

/var/db/dslocal/nodes/Default/users/*.plist           # Local accounts; ShadowHashData (password hashes)
/Library/Keychains/System.keychain                    # System keychain; machine secrets/certs
~/Library/Keychains/login.keychain-db                 # User login keychain; saved creds/tokens
/Library/Keychains/SystemRoots.keychain               # Trust store; rogue roots ⇒ interception
/Library/LaunchDaemons/*.plist                        # System persistence (root); verify owners/paths
/Library/LaunchAgents/*.plist                         # Global user-session persistence
/etc/sudoers , /etc/sudoers.d/*                       # NOPASSWD/wildcards ⇒ instant root
/etc/ssh/sshd_config                                  # Remote entry; weak auth/ciphers/backdoors
/Library/Application Support/com.apple.TCC/TCC.db     # Privacy (TCC) DB; camera/mic/full disk grants

High-risk paths

~/Library/LaunchAgents/*.plist                        # Per-user persistence at login
/System/Library/LaunchDaemons/*.plist                 # Apple daemons (read-only, baseline only)
/System/Library/LaunchAgents/*.plist                  # Apple agents (read-only, baseline only)
/usr/lib/cron/tabs/*                                  # Per-user crontabs (legacy but present)
/etc/periodic/daily|weekly|monthly/*                  # Periodic jobs; simple persistence
/etc/pam.d/*                                          # Auth stack; lax modules ⇒ bypass
/etc/pf.conf                                          # Packet filter rules; egress/ingress control
/Library/Preferences/com.apple.alf.plist              # App firewall config (ALF); disabled ⇒ risk
/Library/Preferences/SystemConfiguration/preferences.plist   # Network services, routes, DNS
/Library/Preferences/SystemConfiguration/com.apple.airport.preferences.plist  # Known Wi-Fi
~/Library/Application Support/com.apple.TCC/TCC.db    # Per-user TCC grants; stealth access
~/Library/Application Support/com.apple.backgroundtaskmanagementagent/backgrounditems.btm  # Login Items
~/Library/Preferences/com.apple.LaunchServices.QuarantineEventsV2  # Gatekeeper quarantine DB
~/.ssh/                                               # User SSH keys/configs; lateral movement

Medium-risk paths

/var/log/* , /Library/Logs/*                          # Legacy logs (some still active); triage
/var/db/diagnostics/*.tracev3                         # Unified logging store; requires log show
/Applications/*                                       # Apps; unsigned/bundled tools
/Library/Application Support/*                         # App data; tokens/configs
~/Library/Application Support/*                        # Per-user app data; browser/Slack/Discord creds
/Library/Extensions/*                                  # Third-party kexts (legacy); root code
~/Library/Safari/ , ~/Library/WebKit/*                # Safari/WebKit data
~/Library/Application Support/Google/Chrome/*         # Chrome profiles (Login Data, Cookies)
~/Library/Application Support/Firefox/Profiles/*      # Firefox profiles (logins.json,key4.db)
/System/Volumes/Data/*                                # Writable data volume (since Catalina)
/private/tmp , /var/tmp                               # Staging; clean/monitor
