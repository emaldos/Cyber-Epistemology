File System Map Important Files

Filesystem map and high-value paths for threats/defense

Critical paths

/etc/shadow                                  # Hashed local passwords; offline cracking ⇒ full host compromise
/root/.ssh/                                  # Root SSH keys; immediate full control if leaked
/etc/ssh/ssh_host_*_key                      # Host identity keys; theft ⇒ MITM/impersonation
/etc/sudoers , /etc/sudoers.d/*              # NOPASSWD/wildcards ⇒ instant root; audit diffs
/etc/ld.so.preload                           # Force-load libs into all processes; stealth persistence
/etc/systemd/system/*.service                # Root services; backdoor via ExecStart payloads
/var/run/docker.sock                         # Root-equivalent Docker control; container breakout ⇒ host root
/etc/ssl/private/*                           # TLS private keys; decryption/impersonation risk
/var/lib/dpkg/status                         # Package truth; poisoned entries hide implants

High-risk paths

/home/*/.ssh/                                # User keys; lateral movement and trust pivot
/etc/ssh/sshd_config                         # Weak auth/ciphers; brute-force and backdoor risk
/etc/netplan/*.yaml                          # Network/DNS routes; persistence via rogue gateways
/etc/resolv.conf , /run/systemd/resolve/*    # DNS control; hijack enables traffic redirection
/usr/local/share/ca-certificates/*           # Added CAs; interception opportunities
/etc/polkit-1/rules.d/*.rules                # Privilege decisions; lax rules ⇒ escalation
/etc/ld.so.conf.d/*                          # Library search path hijack
find / -perm -4000 -type f                   # SUID binaries; misconfigs ⇒ privesc
getcap -r /                                  # File capabilities; elevated syscalls w/o SUID
/var/www/**/.env                             # App secrets (DB/API keys); instant data access
/var/lib/docker/*                            # Images/volumes; harvest secrets from layers
/var/lib/containers/*                        # Podman/container state; creds/tokens

Medium-risk paths

/var/log/auth.log                            # Logins/sudo traces; detect misuse and brute-force
/var/log/syslog , /var/log/kern.log          # System/kernel events; correlate with execution
/var/log/audit/audit.log                     # Syscall/auditd traces; file/exec monitoring
/var/log/journal/                            # Persistent systemd logs; tamper detection
/var/lib/dpkg/ , /var/log/apt/*              # Package inventory/history; spot suspicious installs
/etc/NetworkManager/system-connections/*     # Saved Wi-Fi/802.1x creds; network pivot
/etc/nftables.conf , /etc/ufw/*              # Firewall policy; gaps enable ingress/exfil
/var/www/html                                # Default docroot; common webshell target
/tmp , /var/tmp                              # Payload staging; scan and clean
/boot/*                                      # Kernel/GRUB; bootkits
