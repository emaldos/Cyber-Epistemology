High-Value Paths Important Files

Filesystem map and high-value paths for threats/defense

Critical paths

/data/misc/keystore/                                   # Android Keystore blobs (TEE/StrongBox backed)
/data/system/                                         # System policies; users, gatekeeper, locksettings
/data/system_ce/0/locksettings.db                     # Credential metadata (CE: credential-encrypted)
/data/system_de/0/                                    # DE data (device-encrypted, early boot)
/system/etc/security/cacerts/                         # System trust anchors (read-only)
/data/misc/user/0/cacerts-added/                      # User-added CAs (per-user)
/vendor/etc/selinux/ ; /system/etc/selinux/           # SELinux policies (baseline)
/system/etc/init/ ; /vendor/etc/init/                 # init rc services; boot-time execution (readonly A/B)
/data/adb/modules/ (rooted/Magisk)                    # Root persistence modules

High-risk paths

/data/data/<pkg>/                                     # App sandboxes (DBs, tokens, WebView, files)
/data/user/0/<pkg>/                                   # Multi-user path equivalent of above
/data/misc/wifi/WifiConfigStore.xml                   # Saved Wi-Fi profiles/PSKs (varies by version)
/data/misc/vpn/                                       # VPN profiles/state
/system/etc/hosts ; /etc/hosts                        # Name overrides (system or overlay)
/sdcard/Android/data/<pkg>/                           # App external storage (often sensitive exports)
/data/local/tmp/                                      # Staging for exploits/tools (rooted/dev mode)
/data/tombstones/                                     # Native crash dumps with memory snippets

Medium-risk paths

/sdcard/DCIM/ , /sdcard/Download/ , /storage/emulated/0/  # User files; exfil target
/data/misc/logd/ ; logcat buffers                       # System/app logs (volatile)
/sys/fs/pstore/                                         # Last-kmsg/panic logs after crash
/data/misc/bluetooth/                                   # Pairings/keys (varies by stack)
/data/misc/ethernet/ , /data/misc/net/                  # Network configs/state
/data/system/packages.xml                               # Package inventory/signatures

Note: Access to most paths requires root/adb with appropriate privileges; CE/DE encryption and SELinux confine data at rest and during boot.
