High-Value Paths Important Files

Filesystem map and high-value paths for threats/defense

Critical paths

C:\Windows\System32\config\SAM                        # Local account hashes; offline crack ⇒ escalation
C:\Windows\System32\config\SYSTEM                     # Boot/system keys; needed with SAM for hash extraction
C:\Windows\System32\config\SECURITY                   # LSA secrets context; pointer to DPAPI_SYSTEM
HKLM\SECURITY\Policy\Secrets\*                        # LSA secrets (service creds, cached secrets)
HKLM\SECURITY\Policy\Secrets\DPAPI_SYSTEM             # System DPAPI master key; decrypt machine secrets
HKLM\SOFTWARE\Microsoft\Cryptography\Protect\System   # DPAPI machine key material
%APPDATA%\Microsoft\Credentials\*                     # Credential Manager items (user)
%LOCALAPPDATA%\Microsoft\Credentials\*                # Credential Manager (local)
%LOCALAPPDATA%\Microsoft\Vault\*                      # Windows Vault creds/tokens
%ProgramData%\ssh\ssh_host_*_key                      # OpenSSH host private keys (if enabled)
C:\Windows\System32\drivers\*.sys                     # Kernel drivers; unsigned/malicious = ring0 control

High-risk paths

HKLM\SYSTEM\CurrentControlSet\Services\*              # Service persistence; image paths, autostart
C:\Windows\System32\Tasks\*                           # Scheduled Tasks (XML); beacon/persistence
HKLM\Software\Microsoft\Windows\CurrentVersion\Run*   # Autoruns (machine)
HKCU\Software\Microsoft\Windows\CurrentVersion\Run*   # Autoruns (user)
C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup\   # All-users startup
%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\        # Per-user startup
C:\Windows\System32\wbem\Repository\*                 # WMI repo; event consumer persistence
C:\Windows\System32\drivers\etc\hosts                 # Hostname overrides; redirection
C:\ProgramData\Microsoft\Wlansvc\Profiles\Interfaces\* # WLAN profiles; Wi-Fi keys
%ProgramData%\Microsoft\Network\Connections\Pbk\rasphone.pbk     # VPN phonebook (machine)
%APPDATA%\Microsoft\Network\Connections\Pbk\rasphone.pbk         # VPN phonebook (user)
HKCU\Software\Microsoft\Terminal Server Client\Servers           # RDP target history
C:\Users\<User>\.ssh\*                               # User SSH keys/configs

Medium-risk paths

C:\Windows\System32\winevt\Logs\*.evtx                # Event logs; primary telemetry
C:\ProgramData\Microsoft\Windows Defender\Support\*   # Defender telemetry; bypass attempts
C:\Windows\SoftwareDistribution\*                     # Windows Update cache DB; change timeline
C:\Windows\Prefetch\*.pf                              # Execution traces; triage
C:\Windows\AppCompat\Programs\Amcache.hve             # Program inventory/execution
HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\AppCompatCache  # ShimCache
C:\Windows\System32\sru\SRUDB.dat                     # SRUM network/app patterns
%APPDATA%\Microsoft\Windows\Recent\*                  # Recent files; LNK artifacts
%APPDATA%\Microsoft\Windows\Recent\AutomaticDestinations\*      # Jump Lists
C:\$Recycle.Bin\*                                     # Deleted tools/data
C:\System Volume Information\*                        # VSS snapshots; recover/deleted hives
C:\Users\<User>\AppData\Local\Temp\*                 # User temp staging
C:\Windows\Temp\*                                     # OS temp staging
